# 03 - Revenue Dashboard and Churn Prediction

## Positioning
Business intelligence project for subscription metrics and proactive churn reduction.

## Target market
- SaaS founders and growth teams
- Revenue operations teams

## MVP scope
- Revenue model (MRR, ARR, expansion, contraction)
- Cohort and retention analysis
- Churn risk scoring
- Alerts for high-risk accounts
- Executive dashboard

## Suggested stack
- Data processing: Python + dbt
- Warehouse: BigQuery / Snowflake / PostgreSQL
- BI: Metabase / Superset / Looker Studio
- API: FastAPI for serving predictions

## Commercial packaging
- Starter: metrics dashboard
- Growth: dashboard + churn model
- Enterprise: model monitoring + playbooks + integrations
